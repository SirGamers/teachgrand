Web browsers

Web browsers are like google chrome, Microsoft edge and opera gx.
To use these web browsers, you need to type words into a search engine. Then, you press enter on the keyboard to go to the search results, which are based off of what you typed. For example, if i typed "Tea" into the search bar and pressed enter, it could show me a picture of tea, or a definition of what tea is. You click on the search bar by using your mouse(Remember that?) Using the mouse is fundamental for clicking on links that pop up when the search results appear. Links are search results that appear that usually tell you more about what you searched about. There are a lot of labels that websties use, like .com or .org. .org's are usually owned by legitimate coporations, which makes them trustworthy. .gov's are also extremely trustworthy, as they are owned by the government. Use these websites if possible. Otherwise just use a .com.
