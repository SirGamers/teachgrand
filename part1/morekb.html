Keyboard functions

The keyboard also has multiple different functions, but there are two buttons on it that are important to know.

Shift

Shift is a key on both sides of the keyboard that
Some shift functions on the keyboard include pressing shift+any letter on the keyboard, which capitalizes the letter you choose to type. An example would be shift+a to make A.
holding shift while pressing a number: preforms a special letter, such as an exclamation point, the @ sign, the # sign, the $ sign, the % sign, the ^ sign, the & sign, the * sign, the ( sign and the ) sign.

Ctrl

Some ctrl functions that use can use are ctrl+v, which pastes pictures. Another is ctrl+C, which copies pictures or text. The last fundamental one is ctrl+x, which cuts/deletes whatever is highlighted on the page.
