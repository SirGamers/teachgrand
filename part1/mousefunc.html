Mouse functions

Some more complex right click mouse functions include:

  • select all, which selects all items on the page
  • undo: which undoes your last action
  • Paste: Which pastes what you copied
  • copy: Copies what you choose to, like pictures and text.
  • Cut: Which deletes the selected area.
  • Redo, which we pastes whateevr you deleted.
  • and Emoji: Which accesses the emoji menu (only on some computers.)
